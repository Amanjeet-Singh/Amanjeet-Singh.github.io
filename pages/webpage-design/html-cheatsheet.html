Buttons

Buttons on webpages are elements that not only look good, but perform an action when clicked. A button formatted with a link would take you to a link, a button for submitting information would send information the server and take you a to a link, or as we commonly use them, they can also be used to download things.

Alerts & Modals

Alerts and modals are pieces of a webpage that only show or open when they are triggered by a certain action. These involve JavaScript to function properly and get toggled when the browser detects that you have done something to trigger it. Alerts show on the current page along with the already displayed content, while modals appear on top of the content.
