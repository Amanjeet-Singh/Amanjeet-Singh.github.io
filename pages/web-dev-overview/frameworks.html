Frameworks

Explanation of what frameworks are and examples of popular ones

The languages that we learned about in the last post are vital to developing websites and web applications, and as such, a developer needs to have a vast knowledge of programming and coding in some, if not most, of those languages to do their job efficiently. But even a developer with a wide knowledge of all these langauges might find themselves in trouble trying to code everything from scratch. Today's websites have increasingly dynamic and unique designs and functions, and coding all of it from scratch would require hundreds (or thousands, depending on scale) of hours of coding from many developers and the code would be immense. Frameworks solve this and make writing code and developing websites and application much easier for developers.

What exactly are Frameworks?

Frameworks are tools that make writing code easier. They contain libraries of functions, classes, ids, objects, etc. and each of these 'items' refers to a set of instructions or code that the developer of the framework specifies. So instead of writing out code to make the browser do a specific action, or to design an element in a certain way, we can just assign classes, ids, or call functions, and these are all pre-defined, making the work easier and less time consuming. In most functions, we can even change the definition of these classes, etc. to fit our needs.

Popular Frameworks

Frameworks bring a lot of efficiency to the table, which is why most professional developers use them. One of the most commonly used frameworks for HTML and CSS is Bootstrap, it even uses some javascript to make websites responsive. This website was created with the help of Bootstrap! Some other notable HTML and CSS frameworks are Bulma, Materialize and UIKit.

There are three big frameworks for javascript, these are Angular, Vue.js, and React. There are many more js frameworks out there but these three are the most popular ones currently because of their accessibility and functionality.

Some server-side frameworks include Laravel for PHP, Django and Flask for Python, Ruby on Rails for Ruby, express.js for node.js, and Spring for Java.
