What Are Websites?

An explanation of how websites work

To access a website, we enter a url, or click on a link in our browser window and are taken to a webpage within seconds. There is a lot going on behind the scenes that makes this happen, and the mere fact that this happens in just a few seconds is a testament to how far techonology has progressed since the days it would take minutes to load a single webapge sometimes. There is an extensive and meticulous process that goes behind the simple action of loading a webpage, but first, we must know what a webpage is.

The Webpage

The webpage is what is displayed on our browser when we are taken to a link through a url or hyperlink. A webpage is a file written in HTML, HyperText Markup Language, a markup language, which defines and identifies many elements that are used to structure the website and the content inside of it. When our browser accesses a link, it recieves a file with the extension '.html', commonly known as an html file. The browser then reads the content of this file and arranges the elements of the webpage according to where and how the html file says they should be. (The specifics of how the html language works and shows what is seen on the screen is discussed here). Think of the HTML code as a set of code that tells the browser what it has to display and where, and also, how it should look like. A website is simply a collection of many such webpages that are linked together with each other to form one coherent and universal (usually) experience for the user while they browse through the website.
