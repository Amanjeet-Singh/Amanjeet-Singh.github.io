Frontend Development

A look into frontend development.

As we know, building and publishing websites involves designing the website and making it functional on the browser window of a user. This is known as front-end development, because it involves designing and making the website functional to and for the user.

Front-end developers

Front end developers usually handle the browser side of things. This involves the HTML, CSS and JavaScript involved in designing the website and making it do what it does. In the case of functionality and JavaScript, it has to co-ordinate with the server many times to do what it has to. Frontend developers usually do their part of the job on the issue, and then the back-end devs complete the rest.

What you need to be able to do

Front end developers should be well-versed in their HTML, CSS and JavaScript. These three comprise the majority of the work a front end developer would do. They should not only be good at using the vanilla (no frameworks or libraries included) versions, but also using the frameworks and libraries available for these. Any developer should also learn Git and Tooling, be able to connect to backend APIs, work well with teams, and have a smooth and steady workflow.
